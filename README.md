# WonderPath: Urban Resilience & Neurodiversity Inclusion Platform 🌍🧠

**An Open-Source Civic Science Platform for Urban Resilience Assessment, Digital Technology Prioritization, and Neurodiversity Inclusion.**

## Overview
WonderPath is a decision-support framework designed to bridge the gap between complex urban engineering data and inclusive public policy. Built on international standards (**ISO 37123** and **ISO 37153**) and validated by three peer-reviewed Master's dissertations (UEM/UTFPR), the platform transforms fragmented urban data into **actionable intelligence**.

By integrating **Nature-based Solutions (NbS)**, multicriteria decision methods (**MABAC & TOPSIS**), and citizen science, WonderPath empowers cities to become not only more resilient to climate change but also sensory-friendly for neurodivergent populations (Autism/ADHD).

## Key Features
- **Maturity Assessment:** Measure urban resilience levels based on ISO indicators.
- **Technology Prioritization:** Rank digital transformation investments using the MABAC method.
- **Sensory Heat Maps:** Map urban sensory barriers (noise, heat, flooding) to protect vulnerable citizens.
- **NbS Recommendations:** Automatic suggestions for Green Infrastructure (Sensory Gardens, Linear Parks).
- **Policy Simulator:** Test the impact of public investments before implementation.

## How to Use (User Manual)
### 1. Resilience Maturity (ISO 37153)
1. Navigate to the **Maturity Module**.
2. Input the current values for your city's indicators (e.g., % of green area, drainage capacity).
3. The system will generate a **Maturity Level (1 to 5)** and identify critical gaps.

### 2. Prioritizing Investments (MABAC/TOPSIS)
1. Go to the **Decision Support** tab.
2. Select the technologies you wish to evaluate.
3. Adjust the "Weights" based on local priority (Economic, Social, or Environmental).
4. Click **Calculate** to see the ranking.

### 3. Citizen Science & Sensory Mapping
1. Open the **Interactive Map**.
2. Report areas with high sensory stress (noise or heat).
3. As a manager, visualize the **Sensory Overload Heat Map** to plan Nature-based Solutions.

## Tech Stack
- **Frontend:** React.js / Tailwind CSS
- **Maps:** Leaflet.js / OpenStreetMap API
- **Mathematics:** Custom JS engines for MABAC and Fuzzy TOPSIS.
- **Standards:** ISO 37123:2021 & ISO 37153:2018.
